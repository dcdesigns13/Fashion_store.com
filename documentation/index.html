Getting Started

Installation


  1. Open the html folder to find all the Templates Files
  2. You will need to upload these files to your web server using FTP.
  3. Make sure you upload the required files/folders listed below:
    • html/css - stylesheets folder
    • html/images - images folder
    • html/js - javacripts folder
    • html/fonts - fonts folder
    • html/index.html - index file/homepage
    Other files can be used according to your preferences.
  4. Now you can start adding your content.

Color Schemes


If you want to change your website's color scheme 

     
  1. <!-- Color Schemes -->
    2.  
  2. <link rel="stylesheet" href="css/style.css">
  3. <link rel="stylesheet" href="css/style_red.css">
    4.

If you want to change your website's color scheme

link the stylesheet file css/style_red.css in head section

Make sure you replace the style.css stylesheet in your head with style_red.css stylesheets.

Google Map


For adding your address to Google Map just change coordinates (latitude longitude) in JS code: 

     
  1. <!-- map.js -->
  2. <script>$latitude = 51.5255069,$longitude = -0.0836207,$map_zoom = 14;</script>

Instagram


For adding your instagram account just go to LightWidget generate your Widget replace LightWidget code: 

     
  1. <!-- LightWidget WIDGET -->
  2. <script src="https://cdn.lightwidget.com/widgets/lightwidget.js"></script>
  3. <iframe src="//lightwidget.com/widgets/e00315506df754f3816aa4d695120aeb.html" scrolling="no" allowtransparency="true" class="lightwidget-widget" style="width:100%;border:0;overflow:hidden;"></iframe>
